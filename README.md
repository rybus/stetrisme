# Stetrisme

This Tetris game is a specific version to help people with Amblyopia.


You will need to use anaglyphe glasses to play this game.

## Installation

### Clone the project.

```
git clone git@github.com
```

2. Make sure all SDL dependencies are installed.
- libsdl1.2-dev
- libsdl-ttf2.0-dev
- libsdl-image1.2.dev

3. Compile and run

```
$ make
$ run
```
